## 🏆 MountainCar Results

Training the Deep Q-Network agent on **MountainCar-v0** was more challenging due to sparse rewards.  
- Initially, the agent failed, stuck at -200 reward per episode.  
- After applying **reward shaping**, the agent began to learn momentum strategies.  
- By later episodes, it consistently achieved rewards around **600**, successfully reaching the flag at the top.  

### 🎯 Achievement
This demonstrates that the agent can solve harder environments when guided with improved reward signals.  
It highlights the importance of **reward shaping, hyperparameter tuning, and network design** in Reinforcement Learning research.
